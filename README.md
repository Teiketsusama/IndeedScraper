# IndeedScraper

A Simple Job Scraper Web App using Flask

## Description

Search latest (software engineer) jobs from [au.indeed.com](https://au.indeed.com/)

## Getting Started

### Installing

```
git clone https://github.com/Teiketsusama/IndeedScraper.git
cd IndeedScraper
make install
```

### First Run
```
$ python scraper.py
$ python app.py
```
Go to:

Website: http://localhost:5000
